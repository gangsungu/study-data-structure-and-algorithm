## 스택과 큐

### 스택
+ 후입선출 (LIFO, Last In First Out) 방식으로 동작하는 자료구조
+ 접시쌓기, 실행 취소(Undo) 등의 기능구현에 사용
+ 
    
|연산|설명|시간 복잡도|
|---|---|---|
|push(e)|스택 맨 위에 요소 추가|O(1)|
|pop()|스택 맨 위의 요소 제거|O(1)|
|peek()|스택 맨 위의 요소 확인(제거 X)|O(1)|
|isEmpty()|스택이 비어있는지 확인|O(1)|

### 큐
+ 선입선출 (FIFO, First In First Out) 방식으로 동작하는 자료구조
+ 프린터 대기열, 네트워크 패킷 처리 등에 사용

|연산|설명|시간 복잡도|
|---|---|---|
|enqueue(e)|큐의 맨 뒤에 요소 추가|O(1)|
|dequeue()|큐의 맨 앞에 요소 제거|O(1)|
|peek()|큐의 맨 앞 요소 확인(제거 X)|O(1)|
|isEmpty()|큐가 비어있는지 확인|O(1)|

### 덱
+ Doubled-ended Queue
+ 양쪽에서 삽입과 삭제가 가능한 큐
+ 스택과 큐의 기능 모두 포함

|연산|설명|시간 복잡도|
|---|---|---|
|append(e)|뒤에 요소 추가|O(1)|
|dequeue()|앞에 요소 추가|O(1)|
|pop()|뒤에서 요소 제거|O(1)|
|popleft()|앞에서 요소 제거|O(1)|
|isEmpty()|덱이 비어있는지 확인|O(1)|
